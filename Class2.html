Property Sets

There are 2 property sets(categories) to modify "Content (Content can be text, images, etc.)", which is Text and Font. Text Category has Alignment (text-align) and Decoration (text-decoration).

Font Category has 3 commonly used properties i.e. font-family, font-size, font-weight

Last Defined property (it could be font, size, etc.) will be considered by the browser.

Border is a property used to apply border to the content. 2 commonly used properties are border and border-radius.

Border is a short hand property that has 3 parts size, style (ex - solid, dashed, dotted), and color.

border radius defines the edges of the border in pixels

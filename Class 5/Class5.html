Synchronous Loading - Every tasks will complete only after the previous one is completed.

Asynchronous Loading - The tasks will execute side by side, we dont have to wait for the 1st one to complete.

There are 5 ways to collect user inputs. Text - (Text Box, Text Area), Radio Button, Checkbox, submit button, Normal Button.

Closing Tag can be done inside the starting tag if theres no content between the starting tag and closing tag. ex - (input/)

Radio Input requires defining of Type and Value.

Text Input - 1 Input Type has 1 Label using ID, label has its unique id.

Label has 1 property, its "for".

Restrict user in choosing the available set of options. 1st Set - Only 1 option can be chosen

camel case - 1st letter of 1st word is small and next word's 1st letter is captial - ex - maleOptionGorilla
